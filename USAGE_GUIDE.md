# 网站功能使用指南和故障排除

## 🎯 功能列表和使用方法

### 1. 主题切换功能
**位置**: 右下角橙色圆形按钮（月亮/太阳图标）

**使用方法**:
- 点击按钮即可在深色/浅色主题之间切换
- 主题偏好会自动保存，下次访问时保持

**如果看不到按钮**:
- 清除浏览器缓存（Ctrl+Shift+Delete）
- 硬刷新页面（Ctrl+F5 或 Cmd+Shift+R）
- 检查浏览器控制台是否有JavaScript错误（F12）

---

### 2. 图片灯箱功能
**位置**: Portfolio页面

**使用方法**:
- 在Portfolio页面点击任意项目图片
- 图片会放大显示，带黑色背景
- 点击图片外部或右上角×按钮关闭

**如果点击没反应**:
- 确保在Portfolio页面（portfolio.html）
- 检查图片是否正确加载
- 查看浏览器控制台错误

---

### 3. 打字机效果
**位置**: 首页（index.html）标题

**效果**:
- 页面加载后，"Shengkun Huang"会逐字显示
- 动画速度：150ms/字符

**如果看不到效果**:
- 刷新首页
- 检查是否有JavaScript错误
- 确保打开了index.html而不是其他页面

---

### 4. 技能进度条
**位置**: About页面（about.html）

**显示内容**:
- ArcGIS/QGIS: 90%
- Remote Sensing: 85%
- Python Programming: 75%
- Spatial Analysis: 80%
- Cartography: 85%

**动画效果**:
- 页面加载时进度条从0%动画展开到目标值
- 渐变色彩，带发光效果

**如果看不到进度条**:
- 确保在About页面
- 检查CSS是否正确加载
- 硬刷新页面（Ctrl+F5）

---

### 5. 表单验证
**位置**: Contact页面（contact.html）

**验证规则**:
- 姓名：不能为空
- 邮箱：必须是有效邮箱格式
- 消息：至少10个字符

**实时反馈**:
- 输入框失去焦点时自动验证
- 成功：绿色边框
- 失败：红色边框 + 错误提示

**如果验证不工作**:
- 确保在Contact页面
- 检查浏览器是否支持JavaScript
- 查看控制台错误信息

---

### 6. 滚动进度条
**位置**: 页面顶部

**效果**:
- 青绿色进度条显示阅读进度
- 随滚动实时更新

---

### 7. 返回顶部按钮
**位置**: 右下角（主题切换按钮下方）

**使用方法**:
- 向下滚动300px后自动显示
- 点击平滑滚动到页面顶部

## 🔧 故障排除步骤

### 如果功能不工作，请按以下步骤操作：

#### 步骤1: 清除浏览器缓存
```
Chrome/Edge: Ctrl+Shift+Delete → 选择"缓存的图片和文件" → 清除
Firefox: Ctrl+Shift+Delete → 选择"缓存" → 清除
Safari: Cmd+Option+E
```

#### 步骤2: 硬刷新页面
```
Windows: Ctrl+F5 或 Ctrl+Shift+R
Mac: Cmd+Shift+R
```

#### 步骤3: 检查浏览器控制台
```
1. 按F12打开开发者工具
2. 切换到"Console"标签
3. 刷新页面
4. 查看是否有红色错误信息
5. 如果有错误，截图保存
```

#### 步骤4: 检查文件路径
确保所有文件都在正确位置：
```

├── index.html ✓
├── portfolio.html ✓
├── about.html ✓
├── contact.html ✓
├── assets/
│   ├── css/
│   │   └── main.css ✓
│   └── js/

```

#### 步骤5: 测试单个功能
打开 test.html 文件运行自动测试

## 💡 常见问题

### Q: 为什么我看不到主题切换按钮？
A: 可能是CSS没有正确加载。检查：
1. main.css文件是否存在
2. HTML中是否正确引用了CSS
3. 浏览器控制台是否有404错误

### Q: 点击图片没有反应？
A: 确保：
1. 在portfolio.html页面
2. JavaScript已加载（查看控制台）
3. 图片元素有正确的class

### Q: 表单验证不工作？
A: 检查：
1. 是否在contact.html页面
2. 表单ID是否为"contactForm"
3. 输入框是否有正确的ID

### Q: 进度条不动？
A: 
1. 刷新about.html页面
2. 检查JavaScript是否报错
3. 确认.skill-progress元素存在
