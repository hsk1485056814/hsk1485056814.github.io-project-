# 🕷️ 小蜘蛛功能说明

## 功能介绍

我在你的网站中添加了一个**会自动移动的小蜘蛛**，它会：

### ✨ 特性

1. **自动移动**
   - 小蜘蛛会在页面上随机移动
   - 每3秒改变一次目标位置
   - 平滑的移动动画
   - 会根据移动方向旋转身体

2. **外观特点**
   - 黑色的身体和头部
   - 红色的发光眼睛（会闪烁）
   - 8条会动的腿
   - 顶部有一条蛛丝
   - 立体渐变效果

3. **互动功能**
   - **鼠标悬停**：蜘蛛会暂停移动
   - **点击蜘蛛**：它会瞬间跳到随机位置
   - **离开鼠标**：继续移动

4. **全局存在**
   - 在所有页面都会出现
   - index.html
   - portfolio.html
   - about.html
   - contact.html

## 技术实现

### CSS动画
- 腿部摆动动画（0.5秒循环）
- 眼睛发光效果（2秒循环）
- 平滑的位置过渡

### JavaScript控制
- 随机目标生成
- 向量移动计算
- 角度旋转计算
- 鼠标事件监听

## 自定义选项

如果你想调整蜘蛛的行为，可以修改 `assets/js/main.js` 中的参数：

```javascript
var speed = 2;  // 移动速度（越大越快）

// 在 generateNewTarget 函数中
setInterval(generateNewTarget, 3000);  // 改变目标的间隔时间（毫秒）
```

## 视觉效果

- **深色主题**：蜘蛛会更明显
- **浅色主题**：蜘蛛依然可见
- **移动端**：同样可以工作

## 趣味提示

- 试着追逐蜘蛛，看能否抓住它！
- 快速点击可以让它不断跳跃
- 观察它的腿是如何摆动的
- 注意眼睛的红色光芒

## 注意事项

- 蜘蛛不会干扰页面内容（pointer-events: none）
- 不会影响其他元素的点击
- 性能优化，不会造成卡顿
- 响应式设计，适配各种屏幕
