# Shengkun Huang - GIS Portfolio Website

## 网站介绍
这是一个关于GIS（地理信息系统）的个人作品集网站，展示Shengkun Huang的学术研究、项目作品和专业技能。

## 网站结构

### 页面列表
- **index.html** - 首页（Home）
  - 个人简介
  - 专业背景介绍
  - 快速导航到作品集

- **portfolio.html** - 作品集（Portfolio）
  - 遥感分析项目
  - 空间数据可视化
  - 城市GIS应用
  - 环境监测项目
  - 学术研究
  - 实地调查

- **about.html** - 关于我（About）
  - 个人详细介绍
  - 技能与专长展示
  - 当前研究方向

- **contact.html** - 联系（Contact）
  - 联系表单
  - 合作咨询

## 技术特点

### 前端技术
- HTML5 + CSS3
- JavaScript (jQuery)
- Font Awesome 图标
- 响应式设计

### UI特性
- ✨ 现代化渐变配色方案
- 🎯 平滑的页面过渡动画
- 📊 滚动进度条
- ⬆️ 返回顶部按钮
- 🎨 悬停动效
- 📱 完全响应式设计

### 导航功能
- 侧边栏固定导航菜单
- 当前页面高亮显示
- 所有页面相互链接
- 流畅的用户体验

## 配色方案
- 主色调：深蓝色 (#1a237e, #0d47a1)
- 强调色：青绿色 (#00acc1)
- 背景：渐变色效果
